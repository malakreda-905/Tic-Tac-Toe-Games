# Board Game Suite

A C++ gaming app featuring 8 board game variations built on a
reusable OOP framework, with a GUI. Built for CS213 (OOP) at Cairo University.

## Games Included
- Pyramid Tic-Tac-Toe
- Four-in-a-Row
- 5x5 Tic-Tac-Toe
- Word Tic-Tac-Toe
- Numerical Tic-Tac-Toe
- Misère Tic-Tac-Toe
- Ultimate Tic-Tac-Toe
- SUS

## Features
- Play vs human or random computer player
- GUI built with Qt 
- Extensible OOP framework (Open/Closed Principle)
- Each game inherits from a shared Board/Player base

## Course
CS213 – Object-Oriented Programming | Cairo University
